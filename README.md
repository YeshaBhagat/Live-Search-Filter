# 🔍 Live Search Filter

A simple and interactive **Live Search Filter** built using **HTML, CSS, and JavaScript (DOM manipulation)**.
This project demonstrates how to filter and display data dynamically based on user input without refreshing the page.

---

## 📌 Features

* 🔎 Real-time search filtering
* 🔤 Case-insensitive search
* ⚡ Instant UI updates (no page reload)
* ❌ Displays *"No results found"* when no match exists
* 🎨 Clean and responsive UI design

---

## 🛠️ Technologies Used

* **HTML** – Structure of the web page
* **CSS** – Styling and layout
* **JavaScript** – DOM manipulation and logic

---

## 🚀 How It Works

1. User types in the search box
2. Input is captured using an event listener
3. Text is converted to lowercase for case-insensitive comparison
4. Each list item is checked using a loop
5. Matching items are displayed
6. Non-matching items are hidden
7. If no match is found → "No results found" message is shown
